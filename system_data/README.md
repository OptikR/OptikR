# System Data Folder

This folder contains system-managed content.

## Contents

- **ai_models/** - Downloaded AI models (2-10 GB)
- **cache/** - Performance cache
- **logs/** - Application logs
- **temp/** - Temporary files

## Safe to Delete

This entire folder can be deleted to:
- Clear cache and free space
- Reset to defaults
- Fix corrupted data

The application will recreate it automatically.

## Not Backed Up

This folder does NOT need to be backed up.
All content can be regenerated or re-downloaded.
