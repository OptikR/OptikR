# Learned Data Folder

This folder contains machine learning data that improves over time.

## Contents

- **translations/** - Learned translation dictionaries
- **corrections/** - Learned OCR corrections
- **patterns/** - Learned text patterns

## How It Works

As you use the application, it learns from:
- High-quality translations
- OCR corrections you make
- Text patterns it encounters

This data is saved here and used to improve future results.
