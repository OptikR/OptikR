# User Data Folder

This folder contains user-generated content and settings.

## Contents

- **config/** - User configuration files (user_config.json)
- **learned/** - Learned translations and dictionaries
  - **translations/** - Smart dictionary files per language pair
- **exports/** - Exported data
  - **translations/** - Exported translation files
  - **screenshots/** - Captured screenshots
  - **logs/** - Exported log files
- **custom_plugins/** - User-created plugins
- **backups/** - Configuration backups

## Important

This folder contains YOUR data and should be backed up regularly.

## Backup Recommendations

To backup your settings and learned translations, copy:
- `user_data/config/` - All your settings
- `user_data/learned/` - Your learned dictionaries

## Safe to Edit

You can manually edit configuration files in the `config/` folder, but make backups first.

## Portable Installation

To move OptikR to another computer, copy this entire `user_data/` folder along with the application.
